# neurociencia-del-desarrollo-y-plasticidad-cerebral-datos-longitudinales-0-30-ano
Dataset longitudinal con métricas normativas de desarrollo cerebral desde el nacimiento hasta la edad adulta (0-30 años), por sexo. Variables: grosor cortical, volumen de materia gris, volumen de materia blanca, densidad sináptica, índice de mieliniz
